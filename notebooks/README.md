# Notebooks Overview

This folder contains Google Colab notebooks used for the ISOM 835
Predictive Analytics and Machine Learning individual term project.

## Contents

- `01_EDA.ipynb`
  - Exploratory Data Analysis
  - Data distributions, churn patterns, and feature relationships

- `02_Preprocessing.ipynb`
  - Data cleaning
  - Encoding categorical variables
  - Feature engineering

- `03_Modeling.ipynb`
  - Logistic Regression
  - Random Forest
  - Model evaluation and comparison
